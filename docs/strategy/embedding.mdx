---
title: "Embedding strategies"
description: "Embed a strategy as an iframe with configurable controls and a postMessage API."
---

Any shared strategy can be embedded as an iframe. The embed URL accepts URL parameters and a `playback:*` postMessage protocol for runtime control.

## Quick start

```html
<iframe
	src="https://teambattles.gg/strategy/embed/<shareSlug>?autoplay=1&loop=1"
	width="600"
	height="600"
	frameborder="0"
></iframe>
```

## URL parameters

| Parameter  | Default     | Description                                                |
| ---------- | ----------- | ---------------------------------------------------------- |
| `autoplay` | `1`         | Start playback on mount (`0` or `1`).                      |
| `loop`     | `1`         | Loop back to the first stage at the end (`0` or `1`).      |
| `controls` | `0`         | Show the Play/Stop button (`0` or `1`).                    |
| `chips`    | `0`         | Show stage chips (`0` or `1`).                             |
| `start`    | first stage | Stage id or 1-based index to start on.                     |
| `speed`    | `1`         | Playback multiplier, clamped to `[0.25, 4.0]`.             |
| `branding` | `1`         | Show the "Powered by TeamBattles" link (`0` or `1`).       |

## postMessage API

Send messages to the iframe with `iframe.contentWindow.postMessage({ type, ... }, "*")`:

| Type               | Payload                          | Effect                                       |
| ------------------ | -------------------------------- | -------------------------------------------- |
| `playback:play`    | -                                | Start playback if paused.                    |
| `playback:pause`   | -                                | Stop the in-flight tween.                    |
| `playback:toggle`  | -                                | Flip play/pause.                             |
| `playback:goto`    | `{ stageId? }` or `{ index? }`   | Switch to a stage.                           |
| `playback:setSpeed`| `{ value: number }`              | Update speed multiplier (next transition).   |
| `playback:setLoop` | `{ value: boolean }`             | Update loop setting.                         |

The iframe broadcasts back:

```json
{ "type": "playback:state", "isPlaying": true, "currentStageId": "stg_1", "currentStageIndex": 0 }
```

on every transition and on play/pause.

## Example: external Play button

```html
<button onclick="document.getElementById('strategy').contentWindow.postMessage({ type: 'playback:toggle' }, '*')">
	Play / Pause
</button>
<iframe id="strategy" src="https://teambattles.gg/strategy/embed/abc?controls=0"></iframe>
```

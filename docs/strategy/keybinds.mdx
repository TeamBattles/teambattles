---
title: "Keybinds"
description: "Default keyboard shortcuts and how to customize them."
---

The strategy editor has keyboard shortcuts for all tools and edit actions. Signed-in users can remap any chord via the Keybinds dialog (gear icon in the editor header).

## Defaults

| Action             | Default chord           |
| ------------------ | ----------------------- |
| Select tool        | `V`                     |
| Pen                | `P`                     |
| Line               | `L`                     |
| Arrow              | `A`                     |
| Rectangle          | `R`                     |
| Ellipse            | `O`                     |
| Polygon            | `G`                     |
| Text               | `T`                     |
| Token              | `K`                     |
| Undo               | `Mod + Z`               |
| Redo               | `Mod + Shift + Z`       |
| Copy               | `Mod + C`               |
| Paste              | `Mod + V`               |
| Duplicate          | `Mod + D`               |
| Delete             | `Delete`                |
| Add stage          | `Mod + Enter`           |
| Next stage         | `Mod + Right Arrow`     |
| Previous stage     | `Mod + Left Arrow`      |
| Toggle playback    | `Space`                 |
| Zoom in            | `Mod + =`               |
| Zoom out           | `Mod + -`               |
| Reset zoom         | `Mod + 0`               |

`Mod` = `Cmd` on macOS, `Ctrl` on Windows / Linux.

## Customizing (signed-in only)

<Steps>
	<Step title="Open the Keybinds dialog">
		Click the gear icon in the editor header.
	</Step>
	<Step title="Enter capture mode">
		Click any chord row to enter capture mode.
	</Step>
	<Step title="Press the new chord">
		Press the new chord. Press `Esc` to cancel.
	</Step>
	<Step title="Resolve conflicts">
		If the chord is already bound to another action, you'll be prompted to override.
	</Step>
	<Step title="Save">
		Click **Save** when you're done. Customizations sync across devices.
	</Step>
</Steps>

The **Reset to default** icon next to each row reverts that action. **Reset all to defaults** clears every customization.

## System chords (not rebindable)

`Esc` (contextual cancel) and `Enter` (finalize polygon authoring) are reserved as system chords and do not appear in the dialog.

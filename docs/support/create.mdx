Create a support ticket when you need help.

## Steps to Create

<Steps>
  <Step title="Go to Support">
    Navigate to Support > Create Ticket
  </Step>
  <Step title="Select Entity Type">
    What's your ticket about:
    - **Personal** - Account issues
    - **Organization** - Org management
    - **Team** - Team issues
    - **Match** - Match disputes
    - **Other** - General questions
  </Step>
  <Step title="Select Entity (if applicable)">
    Choose the specific org, team, or match
  </Step>
  <Step title="Select Category">
    Type of issue:
    - Account, Billing, Technical
    - Bug Report, Feature Request
    - Match Dispute, General
  </Step>
  <Step title="Provide Details">
    - **Subject** - Brief summary (5-100 characters)
    - **Description** - Detailed explanation (10-5000 characters)
    - **Images** - Up to 10 screenshots (optional)
  </Step>
  <Step title="Submit">
    Your ticket is created and staff is notified
  </Step>
</Steps>

## Categories

| Category            | When to Use                     |
| ------------------- | ------------------------------- |
| **Account**         | Login, profile, connections     |
| **Billing**         | Payments, subscriptions         |
| **Technical**       | Bugs, errors, performance       |
| **Bug Report**      | Found a bug                     |
| **Feature Request** | Suggest improvements            |
| **Match Dispute**   | Score disputes, rule violations |
| **General**         | Everything else                 |

## Match Dispute Tickets

Special behavior for match disputes:

- **One per match** - Only one open dispute per match
- **Auto-linking** - Match is automatically linked
- **Member access** - Choose who can see the ticket

## Limits

| Setting            | Limit              |
| ------------------ | ------------------ |
| Open tickets       | 5 per user         |
| Subject length     | 5-100 characters   |
| Description length | 10-5000 characters |
| Images per ticket  | 10 total           |
| Image size         | 2 MB each          |

<Warning>
	You can only have 5 open tickets at a time. Close or resolve existing tickets to create new ones.
</Warning>

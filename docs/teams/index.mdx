Teams are the competitive units within organizations. Each team is tied to a specific game and competes in matches.

## What Teams Do

- **Compete** - Challenge other teams and accept matches
- **Track stats** - Win/loss records, XP, streaks
- **Manage rosters** - Active players and bench

Your team profile displays:

- Name, tag, avatar, and banner
- Parent organization
- Game and playlist (game mode)
- Platform and region preferences
- Statistics (wins, losses, XP/points, streaks)
- Active roster and bench

## Team Types

| Type        | XP Display           | Use Case                  |
| ----------- | -------------------- | ------------------------- |
| **Regular** | "150 XP"             | Standard competitive play |
| **League**  | "15.0 pts" (XP / 10) | Organized league play     |

<Note>Both types earn the same XP - only the display format changes.</Note>

## Limits

| Setting  | Limit                              |
| -------- | ---------------------------------- |
| Tag      | 2-5 characters (uppercase)         |
| Name     | Must be unique within organization |
| Tag      | Must be unique within organization |
| Captains | Exactly 1 per team                 |
| Roster   | No hard limit                      |
| Avatar   | 1 MB max                           |
| Banner   | 2 MB max                           |

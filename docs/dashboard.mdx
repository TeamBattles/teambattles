The dashboard is the first thing you see after logging in. It provides an overview of everything happening with your organizations and teams.

## Dashboard Sections

### Your Organizations

See all organizations you belong to at a glance:

- Organization name and avatar
- Your role (Owner, Admin, Manager, Member)
- Quick access to organization pages

### Your Teams

View your teams across all organizations:

- Team name, game, and avatar
- Win/loss record and XP/points
- Quick access to team pages

### Recent Activity

Track what's happening:

- Recent match results
- Roster changes
- Incoming invites and requests

### Upcoming Matches

See scheduled matches:

- Match date and time
- Opponent team
- Current status

### Onboarding Tasks

<Note>
	New users see a checklist of tasks to complete. This section disappears once all tasks are done.
</Note>

The onboarding section guides you through:

- Setting up your profile (avatar, banner, bio)
- Connecting gaming platforms
- Creating or joining an organization and team
- Creating and participating in matches

### Tutorials

The **Tutorials** section lists all available interactive tutorials. Each tutorial walks you through a common task with highlighted UI elements and step-by-step instructions.

- Click any tutorial to start it
- Completed tutorials show a checkmark and can be replayed
- See [Interactive Tutorials](/tutorials) for full details

## Quick Actions

From the dashboard, you can:

- **Create Organization** - Start a new gaming community
- **Create Match** - Challenge other teams
- **View Notifications** - Check recent activity

## Real-Time Updates

Everything on TeamBattles updates in real-time:

- Match scores appear automatically
- Roster changes reflect instantly
- Notifications arrive without refreshing

No need to refresh the page - changes appear as they happen.

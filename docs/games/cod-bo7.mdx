Call of Duty: Black Ops 7 is fully supported with multiple game modes and map pools.

## Requirements

<Warning>
	**Activision connection required.** All team members must link their Activision account.
</Warning>

Link your Activision account:

1. Go to **Settings > Connections**
2. Click **Connect** next to Activision
3. Authorize the connection

## Team Size

| Setting | Value      |
| ------- | ---------- |
| Minimum | 1 player   |
| Maximum | 12 players |
| Default | 4 players  |

## Map Pools

Maps are organized into two pools:

| Pool            | Description                                         |
| --------------- | --------------------------------------------------- |
| **Competitive** | Tournament-standard maps, curated for balanced play |
| **Flex**        | All available maps including casual/pub maps        |

<Note>Some maps may be marked "Coming Soon" if not yet released.</Note>

## Map Selection

During match creation, choose:

- **Selection Type** - Competitive or Flex pool
- **Preference Mode** - Veto (ban maps) or Preferred (pick maps)

### How Map Selection Works

1. Both teams submit their preferences
2. System analyzes both teams' selections
3. Weighted random selection determines final pool
4. Maps preferred by both teams are more likely
5. Final pool fairly represents both preferences

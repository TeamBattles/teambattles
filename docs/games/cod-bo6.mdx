Call of Duty: Black Ops 6 is available in the [Strategy Editor](https://teambattles.gg/strategy/editor) so you can build tactics on every minimap today. Match creation, leagues, and rank tracking for BO6 are still being authored. For a guided walkthrough of the editor, see [Managing strategies](/strategy/managing-strategies).

## Status

| Feature                          | Status                       |
| -------------------------------- | ---------------------------- |
| Strategy editor support          | Available                    |
| Match creation                   | Coming soon                  |
| Leagues                          | Coming soon                  |
| Rank tracking and overlays       | Coming soon                  |

<Note>
	BO6 stays flagged as "Coming Soon" in the match wizard until game modes and playlists are
	authored. You can still draft tactical setups on every released minimap from the strategy editor.
</Note>

## Strategy Editor

The BO6 map roster includes **40 maps**, all available in the strategy editor:

Babylon, Barrage, Blitz, Bounty, Bullet, Dealership, Derelict, Eclipse, Exchange, Extraction, Firing Range, Fringe, Fugitive, Gala, Grind, Hacienda, Haven, Heirloom, Hideout, Jackpot, Lifeline, Lowtown, Nomad, Payback, Pit, Protocol, Racket, Red Card, Rewind, Runway, Scud, Shutdown, Signal, Skyline, Stakeout, Subsonic, Vault, Vorkuta, Warhead, World Motor Dynasty (WMD).

### Overlay Layers

BO6 ships image-overlay layers for every mode the strategy editor exposes:

- Domination zones
- Hardpoint zones
- Search and Destroy zones
- Spawn zones
- Areas
- Headquarters

Marker-based toggles (spawns, POI labels) are not available for BO6 because the upstream guide data does not ship per-map marker coordinates.

<Tip>
	Open the [strategy editor](https://teambattles.gg/strategy/editor), pick **Call of Duty:
	Black Ops 6**, and choose a map to start placing tokens and motion paths.
</Tip>

## What's Coming

The match wizard requires non-empty game modes and playlists, so BO6 stays in coming-soon mode until those are defined:

- Game modes (Hardpoint, Search and Destroy, Control, Domination, Team Deathmatch, and so on)
- Playlists (Core, Hardcore)
- Map pools per mode
- Competitive rules and league configuration
- Activision rank sync

<Warning>
	When BO6 graduates to fully supported, the match wizard will surface modes and playlists
	automatically. Until then, only the strategy editor is wired up.
</Warning>

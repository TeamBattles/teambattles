Counter-Strike 2 supports TeamBattles 5v5 match creation, challenges, league play, lobby codes, score proof, disputes, and the [Strategy Editor](https://teambattles.gg/strategy/editor). Rank tracking is not wired up yet.

## Status

| Feature                    | Status                 |
| -------------------------- | ---------------------- |
| Match creation             | Available              |
| Challenges                 | Available              |
| Leagues                    | Available              |
| Strategy editor support    | Available              |
| Rank tracking and overlays | Not yet wired up       |

<Note>
	CS2 matchmaking is a TeamBattles-managed 5v5 flow. Every rostered player needs a linked
	Steam connection, but TeamBattles does not require CS2 rank sync to create or accept CS2 matches.
</Note>

## Matchmaking

CS2 ships with two 5v5 modes:

- **Premier** - active-duty map pool: Ancient, Anubis, Dust II, Inferno, Mirage, Nuke, and Overpass.
- **Competitive** - full TeamBattles CS2 map pool: Ancient, Anubis, Cache, Dust II, Inferno, Mirage, Nuke, Overpass, Train, and Vertigo.

Teams can use preferred maps, vetoes, or manual map selection according to the match format. The host captain posts the lobby code from the match page when the lobby is ready.

## Strategy Editor

The CS2 map roster includes **10 maps**, all available in the strategy editor:

Ancient, Anubis, Cache, Dust2, Inferno, Mirage, Nuke, Overpass, Train, Vertigo.

### Overlay Layers

The CS2 overlay surface in the strategy editor exposes a single image-overlay layer:

- **buyzones** - shows T and CT buyzone footprints on the active map.

CS2 does not ship marker-based overlays (spawns, POI labels, objective markers) - the engine renders the buyzones overlay only.

<Tip>
	Open the [strategy editor](https://teambattles.gg/strategy/editor), pick
	**Counter-Strike 2**, choose a map, and toggle the **buyzones** layer from the overlay popover to
	see the buy footprint while planning a round.
</Tip>

## What's Not Wired Up Yet

- Rank tracking (no Steam or third-party rank sync)
- Stream overlays for CS2-specific stats

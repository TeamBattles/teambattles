Create a match to challenge other teams. The match creation wizard guides you through three steps.

## Steps to Create

<Steps>
	<Step title="Team & Game">
		Select the team that will be playing, then choose a game:
		- **Call of Duty: Black Ops 7** - 4v4 tactical shooter
		- **Valorant** - 5v5 tactical shooter
		- **League of Legends** - 5v5 MOBA

    	Your team must have the required gaming connection linked for the selected game (e.g., Activision for CoD, Riot Games for Valorant or League of Legends).
    </Step>
    <Step title="Game Configuration">
    	Configure game-specific settings:
    	- **Game mode** - Select from available modes (varies by game)
    	- **Map selection** - Choose map pool type and preference mode (if supported by the game)
    	- **Series length** - Best of 1, 3, 5, or 7
    	- **Roster** - Pick active players and optional bench players
    	- **Game-specific options** - Some games have additional settings (e.g., Draft Pick toggle for League of Legends ARAM)

    	<Tip>Map selection is not available for all games. League of Legends maps are determined by the selected game mode.</Tip>
    </Step>
    <Step title="Match Settings">
    	Finalize the match details:
    	- **Schedule** - Play Now or set a specific date and time
    	- **Restrictions** - Input device and platform restrictions (optional)
    	- **Objectives** - Bonus challenges for extra XP (optional)
    </Step>

</Steps>

Once you have completed all three steps, choose how to proceed:

- **Save as Draft** - Save your match and finish later
- **Create and Publish** - Make the match visible to other teams immediately

## Scheduling Options

| Option    | Description                            |
| --------- | -------------------------------------- |
| Play Now  | Match starts immediately when accepted |
| Scheduled | Set date/time (1 hour to 7 days ahead) |

<Tip>
	Scheduled matches can have an expiration time - if not accepted by then, they are automatically
	cancelled.
</Tip>

## Input Device Options

| Option                | Description       |
| --------------------- | ----------------- |
| All                   | Any input allowed |
| Controller Only       | Controllers only  |
| Keyboard & Mouse Only | KB&M only         |

## Match Objectives

Add optional challenges for bonus experience:

- Complete during the match for +5 XP each
- Both teams can attempt objectives
- Examples: No deaths, headshots only, speed runs

## Publishing

After publishing:

1. Match appears in "Available Matches"
2. Other teams can view and send acceptance requests
3. You review requests and approve an opponent
4. Match moves to "Accepted" status

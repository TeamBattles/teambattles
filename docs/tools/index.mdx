## Tool Cards

Each tool displays:

- **Icon & Name** - Tool branding
- **Tags** - Categories (e.g., "Chrome Extension", "Free")
- **New Badge** - Recently added tools
- **Description** - What it does
- **Features** - Key capabilities
- **Version** - Current version
- **Links** - Download/install buttons
- **Previews** - Screenshot gallery

Stay updated on everything happening with your organizations, teams, matches, leagues, and account.

## Accessing Notifications

Click the **bell icon** in the navigation to view notifications.

## Notification Categories

Notifications are grouped into eight categories with 80 individual notification types in total.

| Category     | Types | What it covers                                                          |
| ------------ | ----- | ----------------------------------------------------------------------- |
| Match        | 22    | Challenges, schedule changes, scores, attendance, disputes, lobby codes |
| Organization | 8     | Invites, join requests, member changes, ownership transfers             |
| Team         | 9     | Invites, join requests, member changes, promotions, demotions           |
| Ticket       | 7     | Support ticket activity and status changes                              |
| Payment      | 2     | Payment receipts and subscription activations                           |
| League       | 23    | Applications, season events, bans, penalties, cooldowns, staff changes  |
| Developer    | 5     | API application status, webhook auto-disable, account suspensions       |
| Other        | 4     | Trophies, announcements, referral signups and milestones                |

### Match Notifications

- Match challenge received / created
- Match challenge accepted / refused
- Match request received / approved / declined / accepted
- Match updated / starting soon / completed / cancelled / forfeited
- Match deleted / unpublished
- Score submitted / needs review / confirmed
- Lobby code updated
- Attendance changed
- Added to roster
- Dispute raised / resolved

### Organization Notifications

- Invite received
- Join request received / approved / declined
- Member accepted / kicked / left
- Ownership transferred

### Team Notifications

- Invite received
- Join request received / approved / declined
- Member accepted / kicked / left
- Member promoted / demoted

### Ticket Notifications

- Ticket created / closed
- New reply on your ticket
- Status changed
- Ticket assigned
- Member added / removed from ticket

### Payment Notifications

- Payment completed
- Subscription activated

### League Notifications

- Player and team applications submitted / approved / denied
- Team removed from league
- Bans and penalties issued, bans revoked
- Cooldowns issued / cleared
- League points changed
- Season started / ended
- Suspended / unsuspended / shutdown
- Staff invited, role changed, removed
- Ticket escalated
- Ownership transferred

### Developer Notifications

- API application submitted / approved / denied
- Developer account suspended
- Webhook auto-disabled

### Other Notifications

- Trophy awarded
- Platform announcements
- Referral signup
- Referral milestone

## Managing Notifications

From the notification panel:

- **View all** - See recent notifications
- **Mark as read** - Clear individual notifications
- **Mark all read** - Clear all at once
- **Click** - Go to related content

## Pending Invites

The **Invites** tab shows:

- Pending organization invites
- Pending team invites
- Accept or decline with one click
- View expiration dates (if set)

## Notification Preferences

Control which notifications you receive and how you get them. You can toggle entire categories or individual notification types.

| Channel | Status      | Description                      |
| ------- | ----------- | -------------------------------- |
| In-App  | Active      | Notifications inside the app     |
| Email   | Coming soon | Notifications sent to your email |
| Push    | Coming soon | Browser push notifications       |

<Note>
	Only the In-App channel is currently active. Email and push toggles appear in your preferences but
	do not deliver yet - they are reserved for upcoming releases.
</Note>

A small set of notifications is treated as critical and cannot be disabled, including payment completion, subscription activation, and developer account suspension. These always appear in the In-App channel so you do not miss billing or account-status changes.

For full details on configuring your preferences, see [Notification Settings](/settings/notifications).

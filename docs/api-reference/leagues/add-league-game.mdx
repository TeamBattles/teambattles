---
title: "Add A League Game"
description: "Add an active game to a league you operate."
openapi: "POST /leagues/{identifier}/games/add"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `games` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to manage league games. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`).

## What's Returned

Adds an active game to the league. On success the response is `{ "success": true, "timestamp": "<ISO 8601>" }`. Adding a game that is already on the league returns `409`.

## Request Body

| Parameter | Type   | Required | Description                     |
| --------- | ------ | -------- | ------------------------------- |
| `gameId`  | string | Yes      | Game ID to add to the league.   |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/games/add \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"gameId": "jh7game123"}'
```

```javascript JavaScript
const leagueSlug = "my-league";

const response = await fetch(`https://teambattles.gg/api/v1/leagues/${leagueSlug}/games/add`, {
	method: "POST",
	headers: {
		Authorization: "Bearer tb_your_api_key",
		"Content-Type": "application/json"
	},
	body: JSON.stringify({
		gameId: "jh7game123"
	})
});

const { success, timestamp } = await response.json();
console.log(success, timestamp);
```

```python Python
import requests

league_slug = 'my-league'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/games/add',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={'gameId': 'jh7game123'}
)

print(response.json())
```

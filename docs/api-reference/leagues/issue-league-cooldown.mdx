---
title: "Issue A League Cooldown"
description: "Issue or update a manual cooldown for a league member."
openapi: "POST /leagues/{identifier}/cooldowns/issue"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `cooldowns` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to issue a cooldown. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`).

## What's Returned

Issues or updates a manual cooldown for a league member. On success the response is `{ "success": true, "cooldownId": "<id>", "timestamp": "<ISO 8601>" }`.

## Request Body

| Parameter       | Type   | Required | Description                                       |
| --------------- | ------ | -------- | ------------------------------------------------- |
| `userId`        | string | Yes      | User ID receiving the cooldown.                   |
| `cooldownHours` | number | Yes      | Cooldown length in hours, from 1 to 8760 (integer). |
| `note`          | string | No       | Optional admin note (up to 500 characters).       |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/cooldowns/issue \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"userId": "jh7user123", "cooldownHours": 48, "note": "Toxicity report"}'
```

```javascript JavaScript
const leagueSlug = "my-league";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/cooldowns/issue`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json"
		},
		body: JSON.stringify({
			userId: "jh7user123",
			cooldownHours: 48,
			note: "Toxicity report"
		})
	}
);

const { success, cooldownId, timestamp } = await response.json();
console.log(success, cooldownId, timestamp);
```

```python Python
import requests

league_slug = 'my-league'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/cooldowns/issue',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={'userId': 'jh7user123', 'cooldownHours': 48, 'note': 'Toxicity report'}
)

print(response.json())
```

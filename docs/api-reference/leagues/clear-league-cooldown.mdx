---
title: "Clear A League Cooldown"
description: "Clear an active league member cooldown early."
openapi: "POST /leagues/{identifier}/cooldowns/{cooldownId}/clear"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `cooldowns` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to clear a cooldown. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The cooldown referenced by `{cooldownId}` must belong to the bound league.

## What's Returned

Clears an active league member cooldown early. This endpoint takes no request body. On success the response is `{ "success": true, "timestamp": "<ISO 8601>" }`.

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/cooldowns/jh7cooldown123/clear \
  -H "Authorization: Bearer tb_your_api_key"
```

```javascript JavaScript
const leagueSlug = "my-league";
const cooldownId = "jh7cooldown123";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/cooldowns/${cooldownId}/clear`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key"
		}
	}
);

const { success, timestamp } = await response.json();
console.log(success, timestamp);
```

```python Python
import requests

league_slug = 'my-league'
cooldown_id = 'jh7cooldown123'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/cooldowns/{cooldown_id}/clear',
    headers={'Authorization': 'Bearer tb_your_api_key'}
)

print(response.json())
```

---
title: "Submit One League Match Map Score"
description: "Submit the score for a single map of a league match."
openapi: "POST /leagues/{identifier}/matches/{matchId}/scores/{mapIndex}"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `scores` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to submit scores. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The match referenced by `{matchId}` must belong to the bound league, and `{mapIndex}` is the zero-based index of the map within the series.

## What's Returned

Submits the score for the single map identified by the `{mapIndex}` path parameter; league-operator scores are auto-confirmed. On success the response is `{ "success": true, "mapIndex": <number>, "scoreStatus": "CONFIRMED" }`.

## Request Body

| Parameter              | Type   | Required | Description                                                                                    |
| ---------------------- | ------ | -------- | ---------------------------------------------------------------------------------------------- |
| `mapId`                | string | Yes      | Identifier of the map that was played.                                                         |
| `creatorTeamScore`     | number | Yes      | Score for the creator team (integer, 0-1000).                                                  |
| `opponentTeamScore`    | number | Yes      | Score for the opponent (accepted) team (integer, 0-1000).                                      |
| `screenshotUrls`       | array  | No       | External screenshot URLs supporting the score. Each must be a public https URL.                |
| `screenshotStorageIds` | array  | No       | Storage IDs from `POST /uploads/image-url`. Preferred over `screenshotUrls` (validated blobs). |
| `playerStats`          | object | No       | Per-player stats keyed by user ID.                                                             |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/matches/jh7match123/scores/0 \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"mapId": "skidrow", "creatorTeamScore": 6, "opponentTeamScore": 3}'
```

```javascript JavaScript
const leagueSlug = "my-league";
const matchId = "jh7match123";
const mapIndex = 0;

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/matches/${matchId}/scores/${mapIndex}`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json"
		},
		body: JSON.stringify({
			mapId: "skidrow",
			creatorTeamScore: 6,
			opponentTeamScore: 3
		})
	}
);

const { success, mapIndex: index, scoreStatus } = await response.json();
console.log(success, index, scoreStatus);
```

```python Python
import requests

league_slug = 'my-league'
match_id = 'jh7match123'
map_index = 0

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/matches/{match_id}/scores/{map_index}',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={'mapId': 'skidrow', 'creatorTeamScore': 6, 'opponentTeamScore': 3}
)

print(response.json())
```

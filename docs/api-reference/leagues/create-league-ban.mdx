---
title: "Create A League Team Ban"
description: "Ban a team from a league you operate."
openapi: "POST /leagues/{identifier}/bans/create"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `bans` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to ban a team. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`).

## What's Returned

Bans a team from the league and removes it from active and pending league participation. On success the response is `{ "success": true, "banId": "<id>", "timestamp": "<ISO 8601>" }`.

## Request Body

| Parameter     | Type    | Required | Description                                                  |
| ------------- | ------- | -------- | ------------------------------------------------------------ |
| `teamId`      | string  | Yes      | Team ID to ban from the league.                              |
| `reason`      | string  | Yes      | Reason for the ban (1-500 characters).                       |
| `isPermanent` | boolean | Yes      | Whether the ban never expires.                               |
| `expiresAt`   | number  | No       | Expiration timestamp for temporary bans, in epoch milliseconds. |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/bans/create \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"teamId": "jh7team123", "reason": "Repeated no-shows", "isPermanent": true}'
```

```javascript JavaScript
const leagueSlug = "my-league";

const response = await fetch(`https://teambattles.gg/api/v1/leagues/${leagueSlug}/bans/create`, {
	method: "POST",
	headers: {
		Authorization: "Bearer tb_your_api_key",
		"Content-Type": "application/json"
	},
	body: JSON.stringify({
		teamId: "jh7team123",
		reason: "Repeated no-shows",
		isPermanent: true
	})
});

const { success, banId, timestamp } = await response.json();
console.log(success, banId, timestamp);
```

```python Python
import requests

league_slug = 'my-league'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/bans/create',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={'teamId': 'jh7team123', 'reason': 'Repeated no-shows', 'isPermanent': True}
)

print(response.json())
```

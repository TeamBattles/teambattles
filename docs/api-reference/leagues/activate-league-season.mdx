---
title: "Activate A League Season"
description: "Move an upcoming league season into the active state."
openapi: "POST /leagues/{identifier}/seasons/{seasonId}/activate"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `seasons` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to activate seasons. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The `{seasonId}` path parameter is the league season ID to activate.

## What's Returned

Activates an upcoming league season. On success the response is `{ "success": true, "timestamp": "<ISO 8601>" }`.

<Note>
	Replays are not deduped. A state guard returns the current state error if the season is no longer upcoming.
</Note>

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/seasons/js5season789/activate \
  -H "Authorization: Bearer tb_your_api_key"
```

```javascript JavaScript
const leagueSlug = "my-league";
const seasonId = "js5season789";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/seasons/${seasonId}/activate`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key"
		}
	}
);

const { success, timestamp } = await response.json();
console.log(success, timestamp);
```

```python Python
import requests

league_slug = 'my-league'
season_id = 'js5season789'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/seasons/{season_id}/activate',
    headers={'Authorization': 'Bearer tb_your_api_key'}
)

print(response.json())
```

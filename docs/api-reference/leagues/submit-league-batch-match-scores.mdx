---
title: "Submit League Map Scores Across Multiple Matches"
description: "Submit map scores for up to 50 league match items in a single request."
openapi: "POST /leagues/{identifier}/matches/batch-scores"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `scores` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to submit scores. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). Every item's `matchId` must belong to the bound league; items targeting a match outside the league fail individually with `error_league_scope_mismatch`.

## What's Returned

Submits map scores across one or more matches in a single call. The top-level response is always HTTP 200: `{ "success": <boolean>, "submitted": [ ... ], "count": <number> }`, where `success` is `true` only when every item was confirmed, each `submitted` entry is `{ "matchId", "mapIndex", "status", "error?" }` in submission order, and `count` is the number of confirmed items. A failing item carries a bare machine-readable error code (e.g. `error_league_scope_mismatch`) and does not abort the others.

<Note>
	A batch may contain at most 50 items. Each item is charged one token against a dedicated
	per-key batch rate-limit bucket; a batch that exceeds the bucket returns `429 error_api_rate_limited`.
</Note>

## Request Body

| Parameter | Type  | Required | Description                                                         |
| --------- | ----- | -------- | ------------------------------------------------------------------- |
| `items`   | array | Yes      | Map scores to submit across one or more matches. 1-50 entries.      |

Each entry in `items` is an object:

| Field                  | Type   | Required | Description                                                                                     |
| ---------------------- | ------ | -------- | ----------------------------------------------------------------------------------------------- |
| `matchId`              | string | Yes      | ID of the match this map score belongs to.                                                      |
| `mapIndex`             | number | Yes      | Zero-based index of the map within the series.                                                  |
| `mapId`                | string | Yes      | Identifier of the map that was played.                                                          |
| `creatorTeamScore`     | number | Yes      | Score for the creator team (integer, 0-1000).                                                   |
| `opponentTeamScore`    | number | Yes      | Score for the opponent (accepted) team (integer, 0-1000).                                       |
| `screenshotUrls`       | array  | No       | External screenshot URLs supporting the score. Each must be a public https URL.                 |
| `screenshotStorageIds` | array  | No       | Storage IDs from `POST /uploads/image-url`. Preferred over `screenshotUrls` (validated blobs).  |
| `playerStats`          | object | No       | Per-player stats keyed by user ID.                                                              |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/matches/batch-scores \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "items": [
      { "matchId": "jh7matchA", "mapIndex": 0, "mapId": "skidrow", "creatorTeamScore": 6, "opponentTeamScore": 3 },
      { "matchId": "jh7matchB", "mapIndex": 0, "mapId": "raid", "creatorTeamScore": 4, "opponentTeamScore": 6 }
    ]
  }'
```

```javascript JavaScript
const leagueSlug = "my-league";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/matches/batch-scores`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json"
		},
		body: JSON.stringify({
			items: [
				{ matchId: "jh7matchA", mapIndex: 0, mapId: "skidrow", creatorTeamScore: 6, opponentTeamScore: 3 },
				{ matchId: "jh7matchB", mapIndex: 0, mapId: "raid", creatorTeamScore: 4, opponentTeamScore: 6 }
			]
		})
	}
);

const { success, submitted, count } = await response.json();
console.log(success, count, submitted);
```

```python Python
import requests

league_slug = 'my-league'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/matches/batch-scores',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={
        'items': [
            {'matchId': 'jh7matchA', 'mapIndex': 0, 'mapId': 'skidrow', 'creatorTeamScore': 6, 'opponentTeamScore': 3},
            {'matchId': 'jh7matchB', 'mapIndex': 0, 'mapId': 'raid', 'creatorTeamScore': 4, 'opponentTeamScore': 6}
        ]
    }
)

data = response.json()
print(data['success'], data['count'], data['submitted'])
```

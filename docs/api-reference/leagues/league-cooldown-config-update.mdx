---
title: "Update League Cooldown Config"
description: "Update a league's member leave and kick cooldown settings."
openapi: "POST /leagues/{identifier}/cooldowns/config/update"
---

## Permission Required

This endpoint requires a `league-operator` API key bound to the league with the `cooldowns` capability set to `write`.

## Membership Required

The API key owner must have an **ADMIN or higher** role in the league, or be a TeamBattles staff member.

## League Identifier

The `{identifier}` path parameter accepts a league slug or Convex league ID.

## Behavior

Cooldown configuration updates control self-leave cooldowns, kick cooldowns, repeat-leave penalties, repeat-leave cooldown multipliers, and the maximum cooldown duration. Hour values must be between `0` and `8760`, and the repeat-leave multiplier must be between `1` and `10`.

Changing this configuration writes a league settings activity entry.

---
title: "Update A League Staff Member Role"
description: "Change the staff role of an existing member on a league you operate."
openapi: "POST /leagues/{identifier}/members/{memberId}/role"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `members` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to change staff roles. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The `{memberId}` path parameter is the league staff membership ID to update.

## What's Returned

Updates the league staff member's role. On success the response is `{ "success": true, "timestamp": "<ISO 8601>" }`.

## Request Body

| Parameter | Type   | Required | Description                                                                     |
| --------- | ------ | -------- | ------------------------------------------------------------------------------- |
| `role`    | string | Yes      | New staff role: `ADMIN`, `MANAGER`, or `MEMBER`. `OWNER` cannot be set via this API. |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/members/jm9member123/role \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"role": "ADMIN"}'
```

```javascript JavaScript
const leagueSlug = "my-league";
const memberId = "jm9member123";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/members/${memberId}/role`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json"
		},
		body: JSON.stringify({
			role: "ADMIN"
		})
	}
);

const { success, timestamp } = await response.json();
console.log(success, timestamp);
```

```python Python
import requests

league_slug = 'my-league'
member_id = 'jm9member123'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/members/{member_id}/role',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={'role': 'ADMIN'}
)

print(response.json())
```

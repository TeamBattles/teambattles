---
title: "Remove League Team"
description: "Remove an approved team from a league."
openapi: "POST /leagues/{identifier}/teams/{leagueTeamId}/remove"
---

## Permission Required

This endpoint requires a `league-operator` API key bound to the league with the `teams` capability set to `write`.

## Membership Required

The API key owner must have a **MANAGER or higher** role in the league, or be a TeamBattles staff member.

## League Identifier

The `{identifier}` path parameter accepts a league slug or Convex league ID.

## League Team ID

The `{leagueTeamId}` path parameter is the approved `leagueTeams` record to remove.

## Request Body

| Parameter | Type   | Required | Description                                       |
| --------- | ------ | -------- | ------------------------------------------------- |
| `reason`  | string | No       | Optional removal reason, up to 500 characters.    |

## Behavior

Removing a team marks it as `REMOVED`, cancels pending matches, forfeits accepted or ready matches for the removed team, notifies the team captain, and writes a league activity entry. In-progress matches are left unchanged.

---
title: "Toggle A League Game"
description: "Activate or deactivate a game on a league you operate."
openapi: "POST /leagues/{identifier}/games/{leagueGameId}/toggle"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `games` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to manage league games. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The `{leagueGameId}` path parameter is the league game link ID.

## What's Returned

Activates or deactivates the league game. On success the response is `{ "success": true, "timestamp": "<ISO 8601>" }`. Deactivating a game cancels its pending, accepted, and ready matches for that league.

## Request Body

| Parameter  | Type    | Required | Description                            |
| ---------- | ------- | -------- | -------------------------------------- |
| `isActive` | boolean | Yes      | Whether the league game should be active. |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/games/jh7lg123/toggle \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"isActive": false}'
```

```javascript JavaScript
const leagueSlug = "my-league";
const leagueGameId = "jh7lg123";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/games/${leagueGameId}/toggle`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json"
		},
		body: JSON.stringify({
			isActive: false
		})
	}
);

const { success, timestamp } = await response.json();
console.log(success, timestamp);
```

```python Python
import requests

league_slug = 'my-league'
league_game_id = 'jh7lg123'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/games/{league_game_id}/toggle',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={'isActive': False}
)

print(response.json())
```

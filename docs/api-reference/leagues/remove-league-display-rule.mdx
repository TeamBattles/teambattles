---
title: "Remove A League Display Rule"
description: "Delete an informational display rule from a league."
openapi: "DELETE /leagues/{identifier}/display-rules/{displayRuleId}"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `display_rules` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to manage display rules. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The `{displayRuleId}` path parameter is the ID of the display rule to remove.

## What's Returned

Removes the informational display rule. On success the response is `{ "success": true, "timestamp": "<ISO 8601>" }`.

<RequestExample>

```bash cURL
curl -X DELETE https://teambattles.gg/api/v1/leagues/my-league/display-rules/jh7rule111 \
  -H "Authorization: Bearer tb_your_api_key"
```

```javascript JavaScript
const leagueSlug = "my-league";
const displayRuleId = "jh7rule111";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/display-rules/${displayRuleId}`,
	{
		method: "DELETE",
		headers: {
			Authorization: "Bearer tb_your_api_key"
		}
	}
);

const { success, timestamp } = await response.json();
console.log(success, timestamp);
```

```python Python
import requests

league_slug = 'my-league'
display_rule_id = 'jh7rule111'

response = requests.delete(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/display-rules/{display_rule_id}',
    headers={'Authorization': 'Bearer tb_your_api_key'}
)

print(response.json())
```

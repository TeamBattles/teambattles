---
title: "Get League Dashboard Stats"
description: "Read aggregate league admin dashboard counts."
openapi: "POST /leagues/{identifier}/dashboard/stats"
---

## Permission Required

This endpoint is an **admin read**. It requires a **league-operator API key bound to this league** with the `leagues.league_admin:read` access permission, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

The `POST` method is used to deliver an authenticated read; this endpoint does not modify the league.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to read dashboard stats. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`).

## What's Returned

Returns aggregate league admin dashboard counts for the bound league as `{ "stats": { ... }, "timestamp": "<ISO 8601>" }`. The `stats` object contains:

| Field                 | Type   | Description                                  |
| --------------------- | ------ | -------------------------------------------- |
| `teamCount`           | number | Number of teams in the league.               |
| `pendingApplications` | number | Team applications awaiting a decision.        |
| `activeSeasons`       | number | Currently active seasons.                     |
| `openTickets`         | number | Open league support tickets.                  |

This endpoint takes no request body.

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/dashboard/stats \
  -H "Authorization: Bearer tb_your_api_key"
```

```javascript JavaScript
const leagueSlug = "my-league";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/dashboard/stats`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key"
		}
	}
);

const { stats, timestamp } = await response.json();
console.log(stats, timestamp);
```

```python Python
import requests

league_slug = 'my-league'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/dashboard/stats',
    headers={'Authorization': 'Bearer tb_your_api_key'}
)

data = response.json()
print(data['stats'], data['timestamp'])
```

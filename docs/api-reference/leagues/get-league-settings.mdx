---
title: "Get League Settings"
description: "Read the full league settings for admin tooling."
openapi: "POST /leagues/{identifier}/settings"
---

## Permission Required

This endpoint is an **admin read**. It requires a **league-operator API key bound to this league** with the `leagues.settings_read:read` access permission, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

The `POST` method is used to deliver an authenticated, filtered read; this endpoint does not modify the league.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to read full settings. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`).

## What's Returned

Returns the full league settings visible to league admins, as `{ "league": { ... }, "timestamp": "<ISO 8601>" }`. The `league` object carries the admin-only settings view for the bound league.

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/settings \
  -H "Authorization: Bearer tb_your_api_key"
```

```javascript JavaScript
const leagueSlug = "my-league";

const response = await fetch(`https://teambattles.gg/api/v1/leagues/${leagueSlug}/settings`, {
	method: "POST",
	headers: {
		Authorization: "Bearer tb_your_api_key"
	}
});

const { league, timestamp } = await response.json();
console.log(league, timestamp);
```

```python Python
import requests

league_slug = 'my-league'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/settings',
    headers={'Authorization': 'Bearer tb_your_api_key'}
)

data = response.json()
print(data['league'], data['timestamp'])
```

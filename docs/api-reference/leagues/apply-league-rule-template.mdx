---
title: "Apply A League Rule Template"
description: "Apply a saved rule template to a league game."
openapi: "POST /leagues/{identifier}/rules/apply-template"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `rules` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to manage enforced rules. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`).

## What's Returned

Applies a saved rule template to one of the league's games, overwriting that game's enforced rules. On success the response is `{ "success": true, "rulesId": "<id>", "timestamp": "<ISO 8601>" }`.

## Request Body

| Parameter    | Type   | Required | Description                            |
| ------------ | ------ | -------- | -------------------------------------- |
| `gameId`     | string | Yes      | Game ID to apply the template to.      |
| `templateId` | string | Yes      | League rule template ID to apply.      |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/rules/apply-template \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"gameId": "jh7game123", "templateId": "jh7tmpl456"}'
```

```javascript JavaScript
const leagueSlug = "my-league";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/rules/apply-template`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json"
		},
		body: JSON.stringify({
			gameId: "jh7game123",
			templateId: "jh7tmpl456"
		})
	}
);

const { success, rulesId, timestamp } = await response.json();
console.log(success, rulesId, timestamp);
```

```python Python
import requests

league_slug = 'my-league'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/rules/apply-template',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={'gameId': 'jh7game123', 'templateId': 'jh7tmpl456'}
)

print(response.json())
```

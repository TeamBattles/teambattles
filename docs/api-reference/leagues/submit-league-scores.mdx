---
title: "Submit League Match Scores In A Batch"
description: "Submit scores for one or more maps of a league match in a single call."
openapi: "POST /leagues/{identifier}/matches/{matchId}/scores"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `scores` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to submit scores. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The match referenced by `{matchId}` must belong to the bound league.

## What's Returned

Submits scores for every map in the `maps` array in a single transaction; league-operator scores are auto-confirmed. The response is `{ "success": <boolean>, "submitted": [ ... ], "count": <number> }`, where `success` is `true` only when every map was confirmed, each `submitted` entry is `{ "mapIndex", "status", "error?" }` in submission order, and `count` is the number of confirmed maps. A failing map is reported in its own entry and does not abort the others.

## Request Body

| Parameter | Type  | Required | Description                                            |
| --------- | ----- | -------- | ------------------------------------------------------ |
| `maps`    | array | Yes      | Map scores to submit. Must contain at least one entry. |

Each entry in `maps` is an object:

| Field                  | Type   | Required | Description                                                                                       |
| ---------------------- | ------ | -------- | ------------------------------------------------------------------------------------------------- |
| `mapIndex`             | number | Yes      | Zero-based index of the map within the series.                                                    |
| `mapId`                | string | Yes      | Identifier of the map that was played.                                                            |
| `creatorTeamScore`     | number | Yes      | Score for the creator team (integer, 0-1000).                                                     |
| `opponentTeamScore`    | number | Yes      | Score for the opponent (accepted) team (integer, 0-1000).                                         |
| `screenshotUrls`       | array  | No       | External screenshot URLs supporting the score. Each must be a public https URL.                   |
| `screenshotStorageIds` | array  | No       | Storage IDs from `POST /uploads/image-url`. Preferred over `screenshotUrls` (validated blobs).    |
| `playerStats`          | object | No       | Per-player stats keyed by user ID.                                                                |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/matches/jh7match123/scores \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "maps": [
      { "mapIndex": 0, "mapId": "skidrow", "creatorTeamScore": 6, "opponentTeamScore": 3 },
      { "mapIndex": 1, "mapId": "raid", "creatorTeamScore": 4, "opponentTeamScore": 6 }
    ]
  }'
```

```javascript JavaScript
const leagueSlug = "my-league";
const matchId = "jh7match123";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/matches/${matchId}/scores`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json"
		},
		body: JSON.stringify({
			maps: [
				{ mapIndex: 0, mapId: "skidrow", creatorTeamScore: 6, opponentTeamScore: 3 },
				{ mapIndex: 1, mapId: "raid", creatorTeamScore: 4, opponentTeamScore: 6 }
			]
		})
	}
);

const { success, submitted, count } = await response.json();
console.log(success, count, submitted);
```

```python Python
import requests

league_slug = 'my-league'
match_id = 'jh7match123'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/matches/{match_id}/scores',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={
        'maps': [
            {'mapIndex': 0, 'mapId': 'skidrow', 'creatorTeamScore': 6, 'opponentTeamScore': 3},
            {'mapIndex': 1, 'mapId': 'raid', 'creatorTeamScore': 4, 'opponentTeamScore': 6}
        ]
    }
)

data = response.json()
print(data['success'], data['count'], data['submitted'])
```

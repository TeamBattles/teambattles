---
title: "Approve League Team Application"
description: "Approve a pending team application for a league."
openapi: "POST /leagues/{identifier}/teams/{leagueTeamId}/approve"
---

## Permission Required

This endpoint requires a `league-operator` API key bound to the league with the `teams` capability set to `write`.

## Membership Required

The API key owner must have a **MANAGER or higher** role in the league, or be a TeamBattles staff member.

## League Identifier

The `{identifier}` path parameter accepts a league slug or Convex league ID.

## League Team ID

The `{leagueTeamId}` path parameter is the pending `leagueTeams` record to approve.

## Behavior

Approving a team marks the pending application as `APPROVED`, records the reviewer, applies league team limits, notifies the team captain, and writes a league activity entry.

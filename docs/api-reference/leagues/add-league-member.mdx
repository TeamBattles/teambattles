---
title: "Add A League Staff Member"
description: "Add or reactivate a staff member on a league you operate."
openapi: "POST /leagues/{identifier}/members/add"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `members` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to add staff. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`).

## What's Returned

Adds or reactivates a league staff member. On success the response is `{ "success": true, "timestamp": "<ISO 8601>" }`.

## Request Body

| Parameter | Type   | Required | Description                                                                     |
| --------- | ------ | -------- | ------------------------------------------------------------------------------- |
| `userId`  | string | Yes      | User ID to add to league staff.                                                 |
| `role`    | string | Yes      | Assignable staff role: `ADMIN`, `MANAGER`, or `MEMBER`. `OWNER` cannot be set via this API. |

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/members/add \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"userId": "jh7abc123def456", "role": "MANAGER"}'
```

```javascript JavaScript
const leagueSlug = "my-league";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/members/add`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json"
		},
		body: JSON.stringify({
			userId: "jh7abc123def456",
			role: "MANAGER"
		})
	}
);

const { success, timestamp } = await response.json();
console.log(success, timestamp);
```

```python Python
import requests

league_slug = 'my-league'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/members/add',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={'userId': 'jh7abc123def456', 'role': 'MANAGER'}
)

print(response.json())
```

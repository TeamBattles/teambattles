---
title: "Update League Branding"
description: "Update a league's public profile branding and metadata."
openapi: "POST /leagues/{identifier}/branding/update"
---

## Permission Required

This endpoint requires a `league-operator` API key bound to the league with the `branding` capability set to `write`.

## Membership Required

The API key owner must have an **ADMIN or higher** role in the league, or be a TeamBattles staff member.

## League Identifier

The `{identifier}` path parameter accepts a league slug or Convex league ID.

## Behavior

Branding updates can change the league name, description, social links, profile theme, avatar storage ID, and banner storage ID. The league name remains globally unique after normalization, and name and description lengths are validated before any write is saved.

When avatar or banner storage IDs are replaced, the previous stored asset is removed from Convex storage.

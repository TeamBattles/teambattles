---
title: "Request League Shutdown"
description: "Request TeamBattles staff review to shut down a league you own."
openapi: "POST /leagues/{identifier}/shutdown/request"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `leagues.league_admin:read-write` access permission, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

This route is **OWNER-only**: only the current league owner may request a shutdown. Convex re-checks current OWNER authority on every call, so a key held by any non-owner staff member is rejected with a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`).

## What's Returned

Requests TeamBattles staff review for league shutdown. On success the response is `{ "success": true, "timestamp": "<ISO 8601>" }`. This does not shut the league down directly - staff-only shutdown execution is not exposed by the public API.

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/shutdown/request \
  -H "Authorization: Bearer tb_your_api_key"
```

```javascript JavaScript
const leagueSlug = "my-league";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/shutdown/request`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key"
		}
	}
);

const { success, timestamp } = await response.json();
console.log(success, timestamp);
```

```python Python
import requests

league_slug = 'my-league'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/shutdown/request',
    headers={'Authorization': 'Bearer tb_your_api_key'}
)

print(response.json())
```

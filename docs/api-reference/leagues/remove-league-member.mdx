---
title: "Remove A League Staff Member"
description: "Deactivate a staff membership on a league you operate."
openapi: "POST /leagues/{identifier}/members/{memberId}/remove"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `members` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to remove staff. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The `{memberId}` path parameter is the league staff membership ID to deactivate.

## What's Returned

Deactivates the league staff membership. On success the response is `{ "success": true, "timestamp": "<ISO 8601>" }`.

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/members/jm9member123/remove \
  -H "Authorization: Bearer tb_your_api_key"
```

```javascript JavaScript
const leagueSlug = "my-league";
const memberId = "jm9member123";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/members/${memberId}/remove`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key"
		}
	}
);

const { success, timestamp } = await response.json();
console.log(success, timestamp);
```

```python Python
import requests

league_slug = 'my-league'
member_id = 'jm9member123'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/members/{member_id}/remove',
    headers={'Authorization': 'Bearer tb_your_api_key'}
)

print(response.json())
```

---
title: "Update A League Match's Lifecycle Status"
description: "Transition a league match between lifecycle states."
openapi: "PATCH /leagues/{identifier}/matches/{matchId}/status"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `scores` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to change a match's status. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The match referenced by `{matchId}` must belong to the bound league.

## What's Returned

Transitions a league match between lifecycle states. On success the response is `{ "success": true, "matchId": "<id>", "previousStatus": "<status>", "newStatus": "<status>", "timestamp": "<ISO 8601>" }`.

<Note>
	Allowed transitions: `READY` -> `IN_PROGRESS` or `CANCELLED`, and `ACCEPTED` -> `CANCELLED`.
	`COMPLETED` is not accepted directly - matches complete automatically once enough map scores
	are confirmed. An invalid transition returns `409 error_match_not_in_valid_state`.
</Note>

## Request Body

| Parameter | Type   | Required | Description                                                  |
| --------- | ------ | -------- | ------------------------------------------------------------ |
| `status`  | string | Yes      | Target status: `IN_PROGRESS` or `CANCELLED`.                 |

<RequestExample>

```bash cURL
curl -X PATCH https://teambattles.gg/api/v1/leagues/my-league/matches/jh7match123/status \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"status": "IN_PROGRESS"}'
```

```javascript JavaScript
const leagueSlug = "my-league";
const matchId = "jh7match123";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/matches/${matchId}/status`,
	{
		method: "PATCH",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json"
		},
		body: JSON.stringify({ status: "IN_PROGRESS" })
	}
);

const { success, previousStatus, newStatus } = await response.json();
console.log(success, previousStatus, newStatus);
```

```python Python
import requests

league_slug = 'my-league'
match_id = 'jh7match123'

response = requests.patch(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/matches/{match_id}/status',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={'status': 'IN_PROGRESS'}
)

print(response.json())
```

---
title: "Remove A League Game"
description: "Remove a game and its non-active configuration from a league."
openapi: "DELETE /leagues/{identifier}/games/{leagueGameId}"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `games` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to manage league games. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The `{leagueGameId}` path parameter is the league game link ID.

## What's Returned

Removes the game from the league and deletes its non-active league configuration. On success the response is `{ "success": true, "timestamp": "<ISO 8601>" }`.

<RequestExample>

```bash cURL
curl -X DELETE https://teambattles.gg/api/v1/leagues/my-league/games/jh7lg123 \
  -H "Authorization: Bearer tb_your_api_key"
```

```javascript JavaScript
const leagueSlug = "my-league";
const leagueGameId = "jh7lg123";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/games/${leagueGameId}`,
	{
		method: "DELETE",
		headers: {
			Authorization: "Bearer tb_your_api_key"
		}
	}
);

const { success, timestamp } = await response.json();
console.log(success, timestamp);
```

```python Python
import requests

league_slug = 'my-league'
league_game_id = 'jh7lg123'

response = requests.delete(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/games/{league_game_id}',
    headers={'Authorization': 'Bearer tb_your_api_key'}
)

print(response.json())
```

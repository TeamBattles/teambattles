---
title: "Create A League Season"
description: "Create an upcoming season for a league game you operate."
openapi: "POST /leagues/{identifier}/seasons/create"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `seasons` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to create seasons. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`).

## What's Returned

Creates an upcoming season for the given league game. On success the response is `{ "success": true, "seasonId": "<id>", "timestamp": "<ISO 8601>" }`.

<Note>
	Supply an optional `Idempotency-Key` request header to make retries safe: an identical key and body replays
	the original season id, while the same key with a different body returns `409 error_idempotency_key_conflict`.
</Note>

## Request Body

| Parameter   | Type   | Required | Description                                  |
| ----------- | ------ | -------- | -------------------------------------------- |
| `gameId`    | string | Yes      | Game ID the season belongs to.               |
| `name`      | string | Yes      | Season name (1-100 characters).              |
| `startDate` | number | Yes      | Season start timestamp in milliseconds.      |
| `endDate`   | number | Yes      | Season end timestamp in milliseconds.        |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/seasons/create \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: 7c1f0a4e-2b3d-4f5a-9c8e-0a1b2c3d4e5f" \
  -d '{"gameId": "call_of_duty", "name": "Spring Split", "startDate": 1717200000000, "endDate": 1719792000000}'
```

```javascript JavaScript
const leagueSlug = "my-league";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/seasons/create`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json",
			"Idempotency-Key": crypto.randomUUID()
		},
		body: JSON.stringify({
			gameId: "call_of_duty",
			name: "Spring Split",
			startDate: 1717200000000,
			endDate: 1719792000000
		})
	}
);

const { success, seasonId, timestamp } = await response.json();
console.log(seasonId, timestamp);
```

```python Python
import requests
import uuid

league_slug = 'my-league'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/seasons/create',
    headers={
        'Authorization': 'Bearer tb_your_api_key',
        'Idempotency-Key': str(uuid.uuid4())
    },
    json={
        'gameId': 'call_of_duty',
        'name': 'Spring Split',
        'startDate': 1717200000000,
        'endDate': 1719792000000
    }
)

print(response.json())
```

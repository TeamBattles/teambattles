---
title: "Submit Or Update League Player Stats For A Map"
description: "Set per-player stats on an existing league match map score."
openapi: "POST /leagues/{identifier}/matches/{matchId}/player-stats"
---

## Permission Required

This endpoint requires a **league-operator API key bound to this league** with the `scores` league capability, plus the `leagueOperatorApi` plan feature (api_pro or higher). A free key receives `403 error_api_feature_required`.

## Access / Membership Rules

You must have a **MANAGER or higher** role in the league (or be a TeamBattles staff member) to update player stats. If you lack the required access, you'll receive a `403` error.

The `{identifier}` path parameter is the league **slug** - the URL-friendly league name (e.g., `my-league`). The match referenced by `{matchId}` must belong to the bound league, and the map score for `mapIndex` must already exist.

## What's Returned

Sets per-player stats on an existing league match map score. On success the response is `{ "success": true, "mapIndex": <number>, "action": "updated" }`.

## Request Body

| Parameter     | Type   | Required | Description                                                                      |
| ------------- | ------ | -------- | -------------------------------------------------------------------------------- |
| `mapIndex`    | number | Yes      | Zero-based index of the map whose stats are being updated.                       |
| `playerStats` | object | Yes      | Per-player stats keyed by user ID. The map score must already exist.             |

<RequestExample>

```bash cURL
curl -X POST https://teambattles.gg/api/v1/leagues/my-league/matches/jh7match123/player-stats \
  -H "Authorization: Bearer tb_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "mapIndex": 0,
    "playerStats": {
      "jh7user123": { "kills": 24, "deaths": 18, "assists": 5 }
    }
  }'
```

```javascript JavaScript
const leagueSlug = "my-league";
const matchId = "jh7match123";

const response = await fetch(
	`https://teambattles.gg/api/v1/leagues/${leagueSlug}/matches/${matchId}/player-stats`,
	{
		method: "POST",
		headers: {
			Authorization: "Bearer tb_your_api_key",
			"Content-Type": "application/json"
		},
		body: JSON.stringify({
			mapIndex: 0,
			playerStats: {
				jh7user123: { kills: 24, deaths: 18, assists: 5 }
			}
		})
	}
);

const { success, mapIndex, action } = await response.json();
console.log(success, mapIndex, action);
```

```python Python
import requests

league_slug = 'my-league'
match_id = 'jh7match123'

response = requests.post(
    f'https://teambattles.gg/api/v1/leagues/{league_slug}/matches/{match_id}/player-stats',
    headers={'Authorization': 'Bearer tb_your_api_key'},
    json={
        'mapIndex': 0,
        'playerStats': {
            'jh7user123': {'kills': 24, 'deaths': 18, 'assists': 5}
        }
    }
)

print(response.json())
```

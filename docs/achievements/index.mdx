The achievement system tracks your progress and showcases your accomplishments.

## Two Purposes

1. **Onboarding Tasks** - Guide new users through platform features
2. **Profile Achievements** - Display accomplishments to other users

## How Achievements Work

- Unlock automatically as you use the platform
- Some have multiple tiers (Bronze, Silver, Gold, Platinum)
- Display on your public profile
- Show rarity compared to other users

## Achievement Display

On your profile, achievements show:

- **Category** - Profile, Organization, Team, Match, Social
- **Unlock Date** - When you earned it
- **Rarity** - Percentage of users who have it
- **Tier Badge** - For tiered achievements

## Rarity Levels

| Percentage | Rarity     |
| ---------- | ---------- |
| > 50%      | Common     |
| 25-50%     | Uncommon   |
| 10-25%     | Rare       |
| 5-10%      | Very Rare  |
| < 5%       | Ultra Rare |

## Privacy

Achievement visibility follows your profile settings:

| Profile Setting | Who Sees Achievements |
| --------------- | --------------------- |
| Public          | Everyone              |
| Limited         | Org/team members only |
| Private         | Only you              |

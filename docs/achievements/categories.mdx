Achievements are organized into five categories.

## Profile Achievements

| Achievement               | Description                       | How to Unlock                       |
| ------------------------- | --------------------------------- | ----------------------------------- |
| Avatar Updated            | Personalized your profile picture | Upload a profile avatar             |
| Banner Updated            | Customized your profile banner    | Upload a profile banner             |
| Bio Updated               | Added a personal bio              | Write something in your bio         |
| Connect a Gaming Platform | Connected a gaming platform       | Link Xbox, PlayStation, Steam, etc. |

<Note>
	Gaming platforms include: Xbox, PlayStation, Steam, Battle.net, Activision, Epic Games, Riot
	Games, Ubisoft, EA. Social platforms (Discord, Twitch, GitHub) don't count.
</Note>

## Organization Achievements

| Achievement          | Description                     | How to Unlock               |
| -------------------- | ------------------------------- | --------------------------- |
| Organization Created | Founded your first organization | Create an organization      |
| Organization Joined  | Became part of a community      | Join any organization       |
| Invite Sent          | Recruited a new member          | Send an organization invite |
| Growing Community    | Built a small community         | Org reaches 5 members       |
| Thriving Community   | Built a medium community        | Org reaches 25 members      |
| Massive Community    | Built a large community         | Org reaches 100 members     |

## Team Achievements

| Achievement   | Description                 | How to Unlock             |
| ------------- | --------------------------- | ------------------------- |
| Team Created  | Started your first team     | Create a team             |
| Team Joined   | Joined a competitive team   | Join any team             |
| Roster Filled | Assembled a complete roster | Fill team's active roster |

## Match Achievements

| Achievement        | Description                  | How to Unlock              |
| ------------------ | ---------------------------- | -------------------------- |
| Match Created      | Created your first challenge | Create and publish a match |
| Match Participated | Competed in your first match | Play in any match          |
| First Victory      | Won your first match         | Win 1 match                |
| Competitor         | Established yourself         | Win 10 matches             |
| Veteran            | Became seasoned              | Win 50 matches             |
| Champion           | Achieved elite status        | Win 100 matches            |

## Social Achievements

| Achievement       | Description                 | How to Unlock                |
| ----------------- | --------------------------- | ---------------------------- |
| Chat Message Sent | Communicated with opponents | Send a message in match chat |
| Friend Invited    | Invited a friend            | Send a platform invite       |

## Tiered Achievements

Some achievements have multiple tiers:

### Organization Members

| Tier   | Threshold   | Badge              |
| ------ | ----------- | ------------------ |
| Bronze | 5 members   | Growing Community  |
| Silver | 25 members  | Thriving Community |
| Gold   | 100 members | Massive Community  |

### Match Wins

| Tier     | Threshold | Badge         |
| -------- | --------- | ------------- |
| Bronze   | 1 win     | First Victory |
| Silver   | 10 wins   | Competitor    |
| Gold     | 50 wins   | Veteran       |
| Platinum | 100 wins  | Champion      |

<Tip>
	Higher tiers include all previous tiers. Earning "Veteran" (50 wins) means you also have
	"Competitor" and "First Victory."
</Tip>

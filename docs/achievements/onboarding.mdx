New users see a task checklist on their dashboard to learn the platform.

## What's Displayed

- Progress indicator (completed vs. total)
- Tasks grouped by category
- Checkmarks for completed tasks
- Direct links to complete pending tasks

## Onboarding Tasks

### Profile

| Task                      | How to Complete                     |
| ------------------------- | ----------------------------------- |
| Update avatar             | Upload a profile picture            |
| Update banner             | Upload a banner image               |
| Update bio                | Write something about yourself      |
| Connect a gaming platform | Link Xbox, PlayStation, Steam, etc. |

### Organization

| Task                  | How to Complete                         |
| --------------------- | --------------------------------------- |
| Create or join an org | Create your own or join an existing one |
| Send an invite        | Invite someone to your organization     |

### Team

| Task                  | How to Complete                      |
| --------------------- | ------------------------------------ |
| Create or join a team | Start a team or join an existing one |
| Fill your roster      | Add players to your active roster    |

### Match

| Task                   | How to Complete           |
| ---------------------- | ------------------------- |
| Create a match         | Publish a match challenge |
| Participate in a match | Play in any match         |

### Social

| Task                | How to Complete                               |
| ------------------- | --------------------------------------------- |
| Send a chat message | Message in match chat (requires active match) |
| Invite a friend     | Send a platform invite                        |

## Completion

- The onboarding section **disappears** once all tasks are done
- Completed tasks remain visible as achievements on your profile
- You can always view achievements in profile settings

<Tip>
	Some tasks require specific conditions - sending a chat message needs an active match with chat
	available.
</Tip>

## Troubleshooting

**"The onboarding section won't disappear"**

- Complete all tasks in each category
- Some tasks have specific requirements (e.g., active match for chat)

**"I can't complete a task"**

- Check the requirements for that task
- Some tasks depend on other tasks (e.g., team creation requires being in an org)

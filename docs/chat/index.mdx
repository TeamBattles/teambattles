Match Chat enables real-time communication between both teams during active matches.

## When Chat is Available

Chat rooms are active when a match has one of these statuses:

- **Accepted** - Both teams confirmed
- **Ready** - Teams checked in
- **In Progress** - Match being played

<Note>
	The chat button is always visible. If you have no active matches, you'll see "No active matches."
</Note>

## Accessing Chat

**Desktop:**

- Click the chat icon in the top navigation
- Or find it in the sidebar footer

**Mobile:**

- Tap the chat icon in navigation
- Opens as a full-screen drawer

## Chat Rooms

Each active match has one chat room where both teams communicate.

**Room List:**

- Shows all your active match chats
- Displays game cover, team names, unread badge
- Sorted by most recent message

**Room Header:**

- Shows match details (game, teams, time)
- Click to navigate to the full match page

Pin chat to float over your screen:

1. Open chat in the popover
2. Click the **Pin** icon
3. Chat becomes a floating window

**Floating Features:**

- Drag to move anywhere
- Resize by dragging corners
- Stays on top of content
- Position and size remembered

To unpin: Click **Unpin** or **Minimize**

## Chat Deletion

<Warning>Chat messages are automatically deleted **48 hours** after a match completes.</Warning>

When a match completes:

1. System announces deletion is scheduled
2. Countdown banner appears
3. You have 48 hours to screenshot important content
4. After 48 hours, all messages and images are deleted

## Offline Behavior

- Messages cached locally for faster loading
- Recent messages appear instantly
- New messages sync when online
- Unread counts persist across sessions

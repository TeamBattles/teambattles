Communicate with your opponents through text, images, and reactions.

## Text Messages

- Type your message and press **Enter** or click **Send**
- Maximum 255 characters per message
- Character counter shows remaining characters

## Images

Share screenshots and images:

- Drag and drop onto the chat
- Or use the attachment button
- Max 10 images per team per match
- Max 2 MB per image
- All images are automatically scanned for NSFW content before uploading

## Replies

Quote and respond to specific messages:

**Desktop:**

1. Hover over a message
2. Click **Reply**
3. Your reply shows the original

**Mobile:**

1. Long-press a message
2. Tap **Reply** in the drawer

## Reactions

React with quick emoji responses:

| Reaction | Emoji |
| -------- | ----- |
| Yes      | ✓     |
| No       | ✗     |
| Good     | 👍    |
| Bad      | 👎    |
| Happy    | 😊    |
| Sad      | 😢    |

Click a reaction to add it. Click again to remove.

## Reporting Messages

If you see inappropriate content:

**Desktop:**

1. Hover over the message
2. Click **Report**

**Mobile:**

1. Long-press the message
2. Tap **Report**

**After reporting:**

- Message flagged for admin review
- You see "This message was reported"
- Hover to view original
- Others see the message until admin action

## System Messages

Chat includes automatic notifications:

| Event           | Message                         |
| --------------- | ------------------------------- |
| Match accepted  | "Match accepted"                |
| Team ready      | "{Team} is ready"               |
| Match started   | "Match started"                 |
| Score submitted | "{Team} submitted score"        |
| Score disputed  | "{Team} disputed Map {N} score" |
| Map picked      | "{Team} picked {Map}"           |
| Map banned      | "{Team} banned {Map}"           |
| Roster updated  | "{Team} updated roster"         |
| Match completed | "Match completed"               |

## Limits

| Setting                | Limit            |
| ---------------------- | ---------------- |
| Characters per message | 255 max          |
| Images per team        | 10 total         |
| Image size             | 2 MB max         |
| Messages per minute    | 5 (rate limited) |

<Note>If rate limited, a countdown timer shows when you can send again.</Note>

## Image Viewing

- Images display as thumbnails
- Click to expand
- Desktop: Click outside or close button to dismiss
- Mobile: Tap anywhere to dismiss

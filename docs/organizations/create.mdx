Create an organization to bring together players and manage teams.

## Steps to Create

<Steps>
	<Step title="Click Create Organization">From your dashboard, click "Create Organization"</Step>
	<Step title="Enter Basic Info">
		- **Name** - 3-50 characters (letters, numbers, spaces only) - **Tag** - 2-6 characters
		(automatically uppercase)
	</Step>
	<Step title="Add Branding (Optional)">
		- **Avatar** - Max 1 MB - **Banner** - Max 2 MB - **Description** - Tell people about your org
	</Step>
	<Step title="Set Visibility">
		- **Public** - Anyone can find and request to join - **Private** - Invite-only
	</Step>
	<Step title="Create">Click "Create" to finish</Step>
</Steps>

## Naming Rules

| Field | Requirements                                             |
| ----- | -------------------------------------------------------- |
| Name  | 3-50 characters, letters/numbers/spaces only             |
| Tag   | 2-6 characters, letters/numbers only (becomes uppercase) |

<Warning>Organization names cannot contain special characters like `@`, `#`, or emoji.</Warning>

## Social Links

After creation, add social links to your profile:

- Twitter/X
- YouTube
- Twitch
- Discord server invite
- Website

Go to your organization's settings to add these links.

Bring new members into your organization through invites or join requests.

## Direct Invites

Invite specific users by username:

<Steps>
	<Step title="Go to Members Tab">Open your organization's Members tab</Step>
	<Step title="Click Invite Member">Opens the invite dialog</Step>
	<Step title="Search for User">Find the user by username</Step>
	<Step title="Select Role">Choose what role they'll have when they join</Step>
	<Step title="Send Invite">The user receives a notification</Step>
</Steps>

## Invite Links

Create shareable links for multiple people:

<Steps>
	<Step title="Go to Members Tab">Open your organization's Members tab</Step>
	<Step title="Click Create Invite Link">Opens the link configuration</Step>
	<Step title="Configure the Link">
		- **Max uses** - How many times the link can be used (default: 1) - **Expiration** - When the
		link stops working (optional)
	</Step>
	<Step title="Share the Link">Copy and share with potential members</Step>
</Steps>

<Tip>Invite links can be permanent if you don't set an expiration.</Tip>

## Join Requests

For **public** organizations, users can request to join:

1. User finds your organization in Discovery
2. User clicks "Request to Join"
3. You receive a notification
4. Review the request in your Members tab
5. Approve or deny with an optional message

<Note>Join requests do not automatically expire. They remain pending until manually reviewed.</Note>

## Managing Invites

### Invite Analytics

The Members page displays summary cards with invite statistics at a glance:

- **Total Invites** - All invites sent
- **Pending** - Awaiting response
- **Accepted** - Successfully joined
- **Declined/Expired** - Declined or past expiration

### Invite History

View a complete history of all invites with status filters:

- **All** - Every invite ever sent
- **Pending** - Awaiting response
- **Accepted** - Successfully joined
- **Declined** - Recipient declined
- **Expired** - Past expiration date
- **Revoked** - Manually cancelled

Each invite displays a color-coded status badge showing its current state.

### Revoke an Invite Link

1. Go to **Members** tab
2. Find the invite in the history list
3. Click the trash icon next to the link

### View Pending Requests

1. Go to **Members** tab
2. Click the **Requests** filter
3. See all pending join requests

## Invite Settings

| Setting    | Default | Notes                  |
| ---------- | ------- | ---------------------- |
| Max uses   | 1       | Can be increased       |
| Expiration | None    | Links can be permanent |

## Troubleshooting

**"My invite link doesn't work"**

- The link may have expired
- The link may have reached max uses
- The link may have been revoked

**"I didn't receive an invite"**

- Check the Invites tab in notifications
- The invite may have expired

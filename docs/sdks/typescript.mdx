title: TypeScript SDK
description: Install and use the official TeamBattles TypeScript SDK to call the developer API.
---

The TypeScript SDK is published to npm as `@teambattles/sdk`. It is generated from the [OpenAPI spec](/developers/api-reference) with [Microsoft Kiota](https://learn.microsoft.com/en-us/openapi/kiota/) and regenerates on every API change.

<Note>TypeScript is a preview SDK - its Kiota runtime dependencies are still pre-release. See [SDKs overview](/sdks) for maturity details.</Note>

## Install

```bash
npm install @teambattles/sdk
```

## Authenticated first call

```ts
import { createTeamBattlesClient } from "@teambattles/sdk";

const client = createTeamBattlesClient(process.env.TEAMBATTLES_API_KEY!);

// List the authenticated user's matches (requires matches.user_matches: read)
const result = await client.user.matches.post({ numItems: 25 });
for (const match of result?.page ?? []) {
	console.log(match.id, match.status);
}
```

## Authentication

The factory wires an `Authorization: Bearer tb_<your-key>` header on every request. Create keys at [Settings > Developer](https://teambattles.gg/settings/developer) and review scopes in the [authentication guide](/developers/authentication). Treat the `error` machine code in error responses - not the `details` message - as the stable programmatic contract.

TeamBattles is where competitive gaming teams create challenges, accept opponents, and compete. Whether you're organizing scrims, running a gaming community, or competing in leagues, TeamBattles provides the tools you need.

## Currently Supported Games

- **Call of Duty: Black Ops 7** - Multiple game modes with competitive and flex map pools
- **Counter-Strike 2** - 5v5 Premier and Competitive matchmaking with strategy editor support
- **Valorant** - Competitive 5v5 with full map pool
- **League of Legends** - Competitive 5v5 with all maps

Additional games (Call of Duty: Black Ops 6 and more) are available in the [Strategy Editor](/strategy/managing-strategies) today, with matchmaking still being authored.

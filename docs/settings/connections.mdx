Manage your login providers and gaming platform connections.

## Login Providers

These let you sign in to TeamBattles:

| Provider   | Status             |
| ---------- | ------------------ |
| Discord    | Connect/Disconnect |
| Twitch     | Connect/Disconnect |
| GitHub     | Connect/Disconnect |
| Battle.net | Connect/Disconnect |

### Linking a Provider

1. Go to **Settings > Connections**
2. Click **Connect** next to the provider
3. Authorize in the popup

### Unlinking a Provider

1. Go to **Settings > Connections**
2. Click **Disconnect** next to the provider

<Warning>You must keep at least one login provider connected to maintain access.</Warning>

## Gaming Platforms

Link these to participate in matches:

| Platform            | Required For         |
| ------------------- | -------------------- |
| Activision          | Call of Duty matches |
| EA                  | EA games             |
| Epic Games          | Fortnite             |
| PlayStation Network | Cross-platform       |
| Riot Games          | Valorant, LoL        |
| Steam               | Steam games          |
| Ubisoft             | Ubisoft games        |
| Xbox Live           | Cross-platform       |

### Connecting a Platform

1. Go to **Settings > Connections**
2. Scroll to **Gaming Platforms**
3. Click **Connect** next to the platform
4. Complete the authorization

<Warning>
	**Activision is required** for Call of Duty matches. All team members need this connection.
</Warning>

### Removing a Platform

1. Go to **Settings > Connections**
2. Click **Disconnect** next to the platform

## Connection Visibility

Control whether connected platforms appear on your public profile. This is separate from your profile visibility setting.

For streaming platforms (Twitch, YouTube, Kick), this toggle also gates the [livestream indicator](/account/livestream) - turning it off hides both the connection and the LIVE badge.

## Verified vs. self-reported

Each connection is tagged based on how it was added:

| Shield | Meaning |
| ------ | ------- |
| Green ShieldCheck | **Verified** - Linked through the platform's OAuth authorization flow. |
| Yellow ShieldAlert | **Self-reported** - Added by typing a username manually. Not proof of ownership. |

The shield is rendered everywhere your connections are shown - on your own settings page, on your public profile, and in places where viewers see your linked platforms. Prefer the OAuth flow when it is available so your connections render as verified.

## Troubleshooting

**"I can't connect my Activision"**

- Ensure you're logged into the correct account
- Try logging out and back in
- Clear browser cache

**"I can't unlink my last provider"**

- Connect a different provider first
- Then unlink the one you want to remove

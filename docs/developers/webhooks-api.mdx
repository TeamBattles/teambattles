---
title: Webhooks
description: Create and manage webhook endpoints to receive real-time TeamBattles events.
---

The **Webhooks** API creates and manages the endpoints that receive real-time events. It
requires the `webhooks.manage` scope and the `webhooks` feature (API Pro or higher). For the
event catalog, payload format, and signature verification, see the
[Webhooks](/developers/webhooks) guide.

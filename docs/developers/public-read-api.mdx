---
title: Public Read API
description: Read public TeamBattles data - games, players, teams, organizations, leaderboards, matches, and leagues.
---

The **Public Read API** exposes public, read-only data: games, players, teams,
organizations, leaderboards, match discovery, and league discovery. Any valid API key with
the relevant read scope can call these endpoints - no special principal or paid plan is
required.

---
title: Game Developer API
description: Submit verified scores and drive the match lifecycle for your game with a game-bound developer key.
---

The **Game Developer API** lets a game server submit verified scores, manage the match
lifecycle, and report rosters and player stats. It requires a **game-developer** key bound
to a specific game, the `game.scores`/`game.lifecycle` scopes, and a paid (API Pro) plan for
writes.

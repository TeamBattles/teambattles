---
title: League Operator API
description: Run a league programmatically - membership, seasons, rules, scores, moderation, and tickets.
---

The **League Operator API** performs league-admin actions: membership and teams, seasons,
rules and config, scores, moderation, tickets, and dashboard reads. It requires a
**league-operator** key bound to a league, the relevant league capability scopes, and the
`leagueOperatorApi` feature - which is gated on the **league owner's** plan (API Pro or
higher).
